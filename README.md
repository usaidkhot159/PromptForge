# 📖 About

PromptForge is a modern AI Prompt Studio that helps users instantly discover, generate, and organize high-quality prompts for Large Language Models (LLMs) like ChatGPT, Gemini, Claude, and more.

Whether you're a developer, designer, student, marketer, or writer, PromptForge provides professionally crafted prompts across multiple categories with an elegant and responsive interface.

### 🎯 Prompt Generation
- Browse hundreds of prompts
- Featured prompt of the day
- Random prompt generator
- Surprise Me button

### 🎭 Personality Modes
Generate prompts from different perspectives.

- Teacher

- Shakespeare

- CEO

- Detective

### ❤️ Prompt Management

- Favorite prompts
- Prompt history
- Copy prompt instantly
- Download as TXT
- Export as PDF

### 🔍 Smart Search

- Search prompts instantly
- Filter using tags
- Difficulty levels
- Language selector

### ⚡ Keyboard Shortcuts

- Space → Generate
- Ctrl + C → Copy
- Ctrl + R → Random
- Ctrl + F → Favorites
- Ctrl + H → History
